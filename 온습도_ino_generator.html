① 필수요소 — Supabase 연결

② 모니터링 조건

현황=실시간 덮어쓰기(짧게), 이력=추이/통계용(길게), 보관일수=이력 자동삭제 기준.

③ WiFi (2.4GHz)

비번 없는 개방망은 비번 칸을 비우세요. 여러 개면 신호 강한 곳에 자동 접속.

④ 보드 등록 (REGISTRY)

MAC은 먼저 ‘MAC 찾기 ino’로 확인 후 입력. 구역은 화면용 짧은 코드(영문/숫자: 1, 2, B_A 등).

1 MAC 찾기 ino (새 보드 MAC 확인용)

WiFi/Supabase 없이 MAC만 화면·시리얼에 표시. 새 보드 등록 전에만 사용.
생성 버튼을 누르세요.

2 모니터링 ino (운영용)

한글 라벨(온도/습도/체감) 포함. 위 입력값이 반영된 완성본.
생성 버튼을 누르세요.

3 Supabase SQL

SQL Editor에 붙여넣고 Run. 테이블·권한·트리거·보관 cron 포함. 재실행 안전.
생성 버튼을 누르세요.
